# Essential Drivers

All the drivers below have to be present at `OC/Drivers`.

**The following drivers are located in the this repo:**
- ApfsDriverLoader.efi
- ext4_x64.efi - Linux driver

**The following drivers come with OC itself:**
- AudioDxe.efi
- OpenCanopy.efi
- OpenHfsPlus.efi
- OpenLinuxBoot.efi
- OpenRuntime.efi
